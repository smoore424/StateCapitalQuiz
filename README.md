# State Capitals Quiz

A game to help students learn and spell the US State Capitals.

()[](/Demo/SCQ1.gif)
()[](/Demo/SCQ2.gif)

Features:
* Accessability support: Dynamic Text and Voice Over
* Dark Mode compatiable
* Persistance, pick up were you left off in the game

Technologies Used:
* Swift 5
* UIKit
* AVAudioPlayer
* Core Animation
